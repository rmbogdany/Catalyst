# Catalyst
Ionic personal scheduling application

## Category

Productivity 

## Description

This is a personal productivity app. Its purpose is to help busy people plan and keep up with all their work, including to-do lists, dates and events, and weekly/long term goals. Its layout is simple so itâ€™s easy to get to each part of the app and you can link locations and view them on a map in order to see where you have to be. 

## Features
  - Goal Tracking
  - Event Scheduling
  - Note-taking
  - Checklists
  - Event Location Tracking
  - Event Notifications

## Cordova Plugins

  - Ionic built in calendar
  - Cordova local notifications plugin
  - Cordova geolocation plugin 
  
## Third-Party APIs
  - Leaflet API 
  - Google Geocoder API
